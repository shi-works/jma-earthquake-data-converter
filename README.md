# jma-earthquake-data-converter
## プログラムについて
- 本プログラムは、気象庁が公開している、[地震月報(カタログ編)の震度データ等](https://www.data.jma.go.jp/eqev/data/bulletin/shindo.html)を読みやすい形式（GISデータ）に変換するプログラムです。
- オープンソースソフトウェアで構築

## デモサイト
- （作成中）

### データの入手
- 地震月報(カタログ編)の震度データ（zip形式）等は、以下の気象庁のサイトから入手します。
- https://www.data.jma.go.jp/eqev/data/bulletin/shindo.html
- なお、一括でダウンロードするPythonスクリプトは以下にあります。
- https://github.com/shi-works/jma-earthquake-data-converter/blob/main/src/shindo_zip_dl.py
